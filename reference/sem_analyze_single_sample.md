# sem_analyze_single_sample

sem_analyze_single_sample

## Usage

``` r
sem_analyze_single_sample(values, thresholds, figure, sample_detail)
```

## Arguments

- values:

  values of methylation

- thresholds:

  threshold to use for comparison

- figure:

  which figure of sample will be analysed: HYPO or HYPER

- sample_detail:

  details of the sample to analyze

## Value

list of lesion count and probe_features count
